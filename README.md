gh-os is a tool that helps you find public information linked to any GitHub account. It scans for emails and basic identifying details that are openly available. This can be helpful if you want to learn more about a GitHub user or check what information they have shared.

The tool works by searching GitHub profiles using public data. It does not hack or access private information.
